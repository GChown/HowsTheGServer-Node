About How's The G

How's The G? was made by Gordie Chown. You can view the source code on Github.

Feel free to email me at Gordie.Chown@gmail.com if you'd like to have a chat.

This website runs on a bunch of free software, like

  • Node.js for server-side computations
  • JQuery to make Javascript easier to work with
  • AngularJS for client-side content
  • Bootstrap for CSS & mobile readiness
  • Let's Encrypt as a free certificate authority
